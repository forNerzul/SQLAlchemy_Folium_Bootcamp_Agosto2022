Bienvenido a Taller Feliz (Recargado)

¿Que es esto?

Esto es un ejercicio de las siguientes tecnologías:

  • Python
  • geopy
  • Flask
  • SQLAlchemy
  • Folium
  • HTML
  • Tailwind CSS
  • Un poquito de JavaScript

¿Que se supone que hace?

Nos permite mediante un formulario seleccionar un punto en el mapa y cargar datos de una ubicación para guardar posteriormente a una base de datos, para mostrar un mapa con todos los puntos cargados, y otro mapa con un filtro de puntos por categoría
